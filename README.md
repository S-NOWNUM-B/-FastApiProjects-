# Репозиторий проектов на FastAPI

Добро пожаловать в мой репозиторий, посвящённый проектам на **FastAPI**! Здесь собрана коллекция образовательных и практических примеров, иллюстрирующих разработку REST API с использованием современных технологий.

## Структура проекта

Репозиторий содержит 20 независимых проектов (FastApiProject_1 - FastApiProject_20), каждый из которых демонстрирует различные аспекты разработки на FastAPI:

- Асинхронная работа с базой данных (PostgreSQL + SQLAlchemy)
- Аутентификация и авторизация (JWT)
- Работа с Redis
- WebSocket соединения
- Интеграция с Celery
- Метрики и мониторинг (Prometheus)
- Логирование
- Тестирование API
- И многое другое

## Технологический стек

Основные технологии, используемые в проектах:

- **FastAPI** (0.104.1) - современный веб-фреймворк для создания API
- **SQLAlchemy** (2.0.23) - ORM для работы с базами данных
- **Pydantic** (2.5.2) - валидация данных и сериализация
- **Alembic** (1.12.1) - миграции базы данных
- **Redis** (5.0.1) - кэширование и очереди
- **Celery** (5.3.4) - асинхронные задачи
- **Prometheus** - мониторинг и метрики
- **Pytest** - тестирование

## Инструкции по запуску

1. **Клонирование репозитория**:
   ```bash
   git clone <адрес_репозитория>
   cd <папка_проекта>
   ```

2. **Настройка окружения**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   .venv\Scripts\activate     # Для Windows
   pip install -r requirements.txt
   ```

3. **Конфигурация**:
   - Создайте файл `.env` в корне проекта
   - Настройте переменные окружения:
     ```
     DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/dbname
     REDIS_URL=redis://localhost:6379
     SECRET_KEY=your-secret-key
     ```

4. **Запуск проекта**:
   ```bash
   uvicorn main:app --reload
   ```

5. **Документация API**:
   - Swagger UI: `http://localhost:8000/docs`
   - ReDoc: `http://localhost:8000/redoc`

## Лучшие практики

- **Безопасность**:
  - Используйте переменные окружения для конфиденциальных данных
  - Применяйте JWT для аутентификации
  - Валидируйте все входящие данные через Pydantic

- **Производительность**:
  - Используйте асинхронные операции
  - Применяйте кэширование через Redis
  - Оптимизируйте запросы к базе данных

- **Тестирование**:
  - Пишите unit-тесты с использованием pytest
  - Тестируйте асинхронный код с pytest-asyncio
  - Используйте фикстуры для тестовых данных

- **Мониторинг**:
  - Настройте логирование через python-json-logger
  - Используйте Prometheus для сбора метрик
  - Отслеживайте производительность API
